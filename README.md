# Installation
## Install via git URL
1. Open the **Package Manager** window
2. Select the **Add package from git URL** option from the `+` dropdown
3. Enter this git url:

### Utility
```
https://github.com/thinhwar123/TW-CustomPackage.git?path=Packages/TW.Utility
```
